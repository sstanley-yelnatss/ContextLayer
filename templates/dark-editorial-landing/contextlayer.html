The problem

AI ships more PRs.
Reviewers still only see the diff.

Teams using Cursor and Copilot merge faster — but the reasoning lives in private chat. What was tried, what failed, what evidence supports the change never reaches the reviewer. Trust drops. Review slows. Compliance teams get nothing auditable.

ContextLayer closes that gap. A persistent reasoning graph on your machine — assumption, action, evidence, conclusion — with hygiene for open loops before they rot. Export selected blocks as PR-ready markdown. Same database over MCP while you work in the IDE.

How it works

Three steps. One continuous loop.

01

Open a workspace

Start with a real goal — a PR you're shipping, a refactor, a bug fix. Use the Agent DevOps template for assumption → action → evidence → conclusion. Security work keeps hypothesis labeling. Data lives in local SQLite on your machine.

Agent DevOps required goal workspace isolation local SQLite timeline view
02

Build the timeline

Add blocks with titles. Record what you assume, what you tried, what you observed, and what you concluded. Structured fields — not a freeform vault. Penetration Testing template uses hypothesis instead of assumption.

assumption action evidence conclusion
03

Export for review

Pick the blocks that explain this change. Export clean markdown and paste into the PR. Reviewers read the curated reasoning receipt; hygiene flags anything still open before you merge.

Why it works

Built on structure.
Driven by governance.

Most tools store what you wrote. ContextLayer stores how understanding evolved — a constrained reasoning state machine you own, with AI as an interface into your graph, not the system of record.

Agent DevOps template — assumption, action, evidence, and conclusion fields tuned for PR reasoning. Security work keeps hypothesis labeling.
Workspace health — orphans, stale items, and dead ends surfaced in a sidebar before unfinished reasoning disappears.
MCP while you chat — Cursor reads and writes the same database as the desktop app. Less copy-paste from chat into your timeline.
Local by default — your graph lives at ~/.contextlayer/graph.db. Your machine. Your data. No cloud sync in this release.
