## Story arc (demo script)

1. **Problem (10s):** AI-assisted PR, empty description, reviewer blind  
2. **Recap their words (10s):** From qual call if personalized  
3. **Flow (40s):** Cursor → MCP checkpoint → hygiene panel → export markdown → paste in PR  
4. **Magic moment (10s):** Reviewer reads *why*, rejected path visible  
5. **CTA (5s):** "3 design partner slots — link in bio / DM"

Use **their repo name** in demo workspace when possible (Pete: personalized demo).

## Production notes
